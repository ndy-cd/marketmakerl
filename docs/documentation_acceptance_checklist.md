# Documentation Acceptance Checklist

Минимальный чеклист качества документации.

- [ ] `README.md` короткий и понятный
- [ ] `docs/PROJECT_GUIDE.md` содержит каноническое описание
- [ ] `docs/DOCS_INDEX.md` объясняет куда идти по ролям
- [ ] Нет дублирования больших блоков между файлами
- [ ] Команды в документации запускаются как есть
- [ ] Ограничение `PAPER_ONLY=1` явно отражено
